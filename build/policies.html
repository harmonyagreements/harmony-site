Policies

This page is a DRAFT for public consideration.

Project Harmony encourages FOSS projects to use the Template Agreements as drafted but permits their modification for a particular FOSS project. We have drafted the Template Agreements to be useful for a wide range of FOSS projects, and they may be used at no cost subject to compliance with the terms below. We encourage FOSS projects to use the unmodified [Do we need to explain the difference between selecting from the existing options and modifications?] Template Agreements because the FOSS project may then use the Project Harmony trademarks and it will ease legal review as the Template Agreements become more familiar to potential contributors.

However, if a FOSS project determines that it needs to modify the Template Agreements, we encourage the FOSS project to obtain legal advice because the provisions of the Template Agreements are inter-related and a single change can have multiple effects.
